Control Systems · Classical Control

Lead-Lag Compensator Design for TRMS

COMPLETED

Problem — Designed a Lead-Lag compensator for the pitch channel of the Twin Rotor Multi-Input Multi-Output System (TRMS). The objective was to improve transient response, reduce overshoot, and stabilize the system using the Root Locus technique.

Approach — Converted the TRMS state-space model into a transfer function, analyzed the open-loop dynamics using Root Locus, and designed an optimized Lead-Lag controller in MATLAB. The controller parameters were selected through controller optimization and validated using Root Locus, step response, pole-zero map, and Bode plot analysis.

Result — Achieved a stable closed-loop controller with approximately 2.38% overshoot while maintaining system stability. The project demonstrates the application of classical control theory and MATLAB for controller design of higher-order dynamic systems.

Control Systems · State-Space Analysis

Analysis of the Twin Rotor MIMO (TRMS) System

COMPLETED

Problem — Implemented and analyzed the Twin Rotor Multi-Input Multi-Output (TRMS) system using classical control and state-space techniques to investigate its dynamic behavior, stability, controllability, observability, and structural characteristics.

Approach — Developed the sixth-order TRMS state-space model in MATLAB, derived its transfer function matrix, and performed root locus analysis, transient response evaluation, eigenvalue analysis, controllability and observability tests, PBH criterion, observability Gramian, Hankel singular value analysis, minimal realization, and hidden-state investigation.

Result — Demonstrated that the TRMS model is open-loop stable and completely controllable while identifying one internally stable but unobservable state. The sixth-order system was successfully reduced to an equivalent fifth-order realization without altering its observable input-output behavior.
